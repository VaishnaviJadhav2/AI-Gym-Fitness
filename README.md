# 🏋️ FitAI – AI Gym & Fitness Assistant

An AI-powered Gym & Fitness Web Application that helps users achieve their fitness goals through personalized workout plans, diet recommendations, BMI calculation, gym finder, and an intelligent AI chatbot.

---

## 🚀 Features

- 🤖 AI Fitness Chatbot
- 💪 Personalized Workout Recommendations
- 🥗 Diet Planner
- 📊 BMI Calculator
- 📍 Gym Finder
- 🔐 User Authentication
- 📈 Performance Tracking
- 📱 Fully Responsive Design

---

## ✨ Key Modules

### 🧠 AI Chatbot
Provides AI-powered fitness guidance and answers user fitness-related questions.

### 💪 Workout Planner
Generates personalized workout plans based on fitness goals.

### 🥗 Diet Recommendation
Suggests healthy diet plans according to user preferences.

### 📍 Gym Finder
Helps users locate nearby gyms.

### 📊 BMI Calculator
Calculates BMI and categorizes fitness status.

### 👤 User Authentication
Secure Login and Registration System.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Vaishnavijadhav07/AI-Gym-Fitness.git
```

### Navigate into Project

```bash
cd AI-Gym-Fitness
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
npm start
```

### Install Backend Dependencies

```bash
cd backend
pip install -r requirements.txt
python app.py
```
